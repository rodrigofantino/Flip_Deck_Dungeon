FLIP DECK DUNGEON
GAME DESIGN DOCUMENT (GDD)

========================
1. VISION GENERAL
========================
Flip Deck Dungeon es un roguelike de cartas con combates por turnos donde todas las cartas
son entidades vivas con HP, dano, nivel y traits.

Plataforma: PC (Steam)
Motor: Godot 4.x
Duracion de runs: corta / media
Enfoque: decisiones claras, progresion persistente, rejugabilidad.

========================
2. PILARES DE DISENO
========================
- Cartas como entidades vivas
- Una sola fuente de verdad
- Runs cortas pero profundas
- Progresion clara y acumulativa
- Riesgo vs recompensa

========================
3. CORE FANTASY
========================
El jugador encarna a un heroe representado por una carta.
Los enemigos tambien son cartas que se enfrentan una a una.
Cada combate debilita o fortalece la run.

========================
4. LOOP PRINCIPAL
========================
MACRO LOOP:
- Inicio de run
- Combates secuenciales
- Recompensas
- Level up y traits
- Derrota o victoria
- Retorno al meta

MICRO LOOP:
- Robar enemigo
- Combate
- Resolucion
- Limpieza de estado

========================
5. SISTEMA DE CARTAS
========================
Tipos:
- Heroe
- Enemigo

Cada carta tiene:
- ID
- Definicion base
- Nivel
- HP actual / maximo
- Dano
- Traits

========================
6. SISTEMA DE COMBATE
========================
Combate por turnos:
- Turno heroe
- Turno enemigo
- Aplicacion de dano
- Chequeo de muerte
- Fin de ronda

========================
7. SISTEMA DE TRAITS
========================
Traits como modificadores de reglas.
Se aplican mediante hooks:
- Inicio de combate
- Ataque
- Dano
- Muerte

========================
8. TUTORIAL
========================
- Run guiada
- Sin RNG duro
- Ensenia todo el juego
- Cartas de tutorial no se pierden

========================
9. META PROGRESION
========================
- Oro persistente
- Sobres (boosters)
- Coleccion
- Boton Continue para continuar run guardada

========================
10. COLECCION
========================
Libro de cartas
Filtros por rareza y bioma

========================
11. POPUPS Y FLOW DE VICTORIA/DERROTA
========================
- Derrota: popup con boton para volver al menu
- Victoria: popup con celebracion, oro obtenido y boton para volver al menu
- La victoria no ofrece traits ni level up si no hay enemigos restantes

========================
12. PRINCIPIOS
========================
- Sistemas antes que contenido
- Datos > UI
- Claridad > complejidad
